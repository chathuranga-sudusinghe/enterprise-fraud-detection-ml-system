# AI Usage Disclosure

This project was developed with selective assistance from AI tools such as ChatGPT and Codex. These tools supported parts of the engineering workflow but did not independently design, build, validate, or approve the project.

## How AI Tools Were Used

AI assistance was used for:

- Project planning and architecture discussion
- Documentation drafting and refinement
- Selected code generation support
- Test and validation planning
- GitHub workflow and repository maintenance guidance
- Refactoring suggestions and review support

## Human Review and Ownership

The maintainer remains responsible for:

- Final project direction
- Architecture decisions
- Code correctness
- Security review
- Testing and validation
- Git history and release decisions
- Public documentation accuracy

## Project Standards

- AI-generated or AI-suggested changes must not be accepted blindly.
- Changes must remain small, focused, and reviewable.
- Secrets, credentials, datasets, model artifacts, and large generated files must not be committed.
- Relevant tests and checks must be run where applicable.
- Changes must follow [CONTRIBUTING.md](CONTRIBUTING.md) and the maintainer-led roadmap.

## Transparency

AI is used as an engineering productivity tool, not as a replacement for human understanding, review, or responsibility.
